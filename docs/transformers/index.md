# Transformers教程

Hugging Face Transformers是最流行的预训练模型库，支持BERT、GPT、LLaMA等主流模型。

## 为什么学习Transformers？

- **预训练模型**：数千个开箱即用的模型
- **统一API**：一致的接口调用不同模型
- **生态完善**：Datasets、Tokenizers配套工具
- **社区活跃**：最新模型第一时间支持

## 核心概念

- **预训练模型**：AutoModel、AutoTokenizer
- **Tokenizer**：文本到token的转换
- **Trainer**：训练循环封装
- **Pipeline**：快速推理接口
