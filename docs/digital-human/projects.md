# 实践项目

通过实际项目练习，快速提升你的数字人创作能力！

## 入门项目

### 项目一：制作口播视频

**难度**：⭐  
**预计时间**：30分钟  
**学习目标**：掌握基础数字人视频制作

**项目描述**：
使用预制数字人制作一个30秒的口播视频。

**步骤指引**：
1. 选择一个数字人平台（推荐：剪映、HeyGen）
2. 选择一个预制数字人形象
3. 编写30秒的口播文案
4. 生成视频并下载

**提交要求**：
- 一个30秒的口播视频
- 简短的文案说明

---

### 项目二：克隆自己的数字人

**难度**：⭐⭐  
**预计时间**：1小时  
**学习目标**：掌握数字人克隆技术

**项目描述**：
克隆一个自己的数字人形象，并制作一段视频。

**步骤指引**：
1. 准备拍摄环境（光线充足、背景简洁）
2. 按平台要求录制训练视频
3. 等待克隆完成
4. 使用克隆形象制作视频

**提交要求**：
- 克隆的数字人视频
- 拍摄环境说明

## 进阶项目

### 项目三：产品介绍视频

**难度**：⭐⭐⭐  
**预计时间**：2小时  
**学习目标**：掌握商业内容创作

**项目描述**：
制作一个1分钟的产品介绍视频，包含数字人口播和产品展示。

**步骤指引**：
1. 选择一个产品
2. 编写产品介绍脚本
3. 准备产品图片或视频素材
4. 制作数字人口播部分
5. 剪辑合成完整视频

**提交要求**：
- 1分钟产品介绍视频
- 脚本和素材说明

---

### 项目四：直播带货脚本

**难度**：⭐⭐⭐⭐  
**预计时间**：3小时  
**学习目标**：掌握直播内容策划

**项目描述**：
策划并制作一个30分钟的直播带货脚本和演示视频。

**步骤指引**：
1. 选择带货产品
2. 编写30分钟直播脚本
3. 设置数字人直播场景
4. 录制演示片段
5. 整理话术库

**提交要求**：
- 完整直播脚本
- 演示视频片段
- 话术库文档

## 高级项目

### 项目五：企业培训课程

**难度**：⭐⭐⭐⭐⭐  
**预计时间**：8小时  
**学习目标**：掌握企业级内容制作

**项目描述**：
制作一套完整的企业培训视频课程，包含3-5个视频。

**步骤指引**：
1. 确定培训主题和大纲
2. 编写详细脚本
3. 设计数字人形象和场景
4. 批量制作视频
5. 添加字幕和特效
6. 整理课程资料

**提交要求**：
- 3-5个培训视频
- 课程大纲文档
- 配套资料
