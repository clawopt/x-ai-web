# LightGBM教程

LightGBM是微软开源的梯度提升框架，采用Leaf-wise生长策略，训练速度快、内存占用低。

## 为什么学习LightGBM？

- **训练速度快**：比XGBoost快数倍
- **内存占用低**：适合大规模数据
- **精度高**：Leaf-wise生长策略
- **支持GPU**：GPU加速训练

## 核心特性

- **Leaf-wise生长**：按叶子分裂，减少过拟合
- **直方图算法**：加速特征分裂查找
- **类别特征支持**：原生支持类别变量
- **分布式训练**：支持多机多卡
