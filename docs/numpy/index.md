# NumPy教程

NumPy是Python数值计算的基石，提供了高性能的多维数组对象和相关工具。

## 为什么学习NumPy？

- **数值计算基石**：几乎所有Python科学计算库都基于NumPy
- **高性能**：底层C实现，比纯Python快100倍以上
- **向量化运算**：避免循环，代码简洁高效
- **广播机制**：不同形状数组间的自动运算

## 核心概念

- **ndarray**：N维数组对象
- **向量化运算**：批量操作，无需循环
- **广播机制**：自动扩展数组形状
- **索引与切片**：灵活的数据访问
