# LangChain教程

LangChain是LLM应用开发的首选框架，提供Chain、Agent、Memory等核心组件。

## 为什么学习LangChain？

- **应用框架**：快速构建LLM应用
- **组件丰富**：Chain、Prompt、Memory、Agent
- **生态完善**：与各种LLM和向量库集成
- **生产就绪**：LangServe部署支持

## 核心概念

- **Chain链**：组合多个组件的工作流
- **Prompt模板**：动态提示词构建
- **Memory记忆**：对话历史管理
- **Agent智能体**：自主决策执行
