# C++基础教程

C++是系统编程的核心语言，是CUDA、TensorRT等高性能计算库的基础。

## 为什么学习C++？

- **高性能**：接近底层，执行效率极高
- **系统编程**：操作系统、游戏引擎首选
- **AI基础设施**：PyTorch、TensorFlow底层都是C++
- **CUDA基础**：GPU编程需要C++基础

## 核心内容

- **语法基础**：变量、控制流、函数
- **指针与引用**：内存管理核心概念
- **面向对象**：类、继承、多态
- **STL标准库**：容器、算法、迭代器
