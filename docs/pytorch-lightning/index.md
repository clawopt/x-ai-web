# PyTorch Lightning教程

PyTorch Lightning是PyTorch的工程化封装，提供简洁的API和自动化训练流程。

## 为什么学习PyTorch Lightning？

- **工程化封装**：简化训练循环代码
- **自动分布式**：轻松实现多GPU训练
- **日志管理**：集成TensorBoard、W&B等
- **Checkpoint**：自动保存和恢复训练

## 核心概念

- **LightningModule**：模型封装基类
- **Trainer**：训练器，自动管理训练流程
- **DataModule**：数据模块，封装数据加载
- **Callbacks**：回调函数，自定义训练行为
