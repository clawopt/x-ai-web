# PyTorch教程

PyTorch是当前最流行的深度学习框架，以动态图和易用性著称。

## 为什么学习PyTorch？

- **动态计算图**：调试方便，代码直观
- **Pythonic风格**：与NumPy无缝衔接
- **社区活跃**：论文复现首选框架
- **生态丰富**：Transformers、Lightning等扩展

## 核心概念

- **Tensor张量**：GPU加速的多维数组
- **Autograd自动微分**：自动计算梯度
- **nn.Module**：神经网络模块基类
- **DataLoader**：高效数据加载
