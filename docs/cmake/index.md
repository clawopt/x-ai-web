# CMake教程

CMake是跨平台构建工具，是C++和CUDA项目的标准构建系统。

## 为什么学习CMake？

- **跨平台构建**：一套配置，多平台编译
- **依赖管理**：find_package自动查找库
- **CUDA支持**：原生支持CUDA项目构建
- **工业标准**：大型C++项目首选

## 核心内容

- **基础配置**：CMakeLists.txt编写
- **目标管理**：add_executable、add_library
- **依赖查找**：find_package、target_link_libraries
- **CUDA支持**：enable_language(CUDA)
