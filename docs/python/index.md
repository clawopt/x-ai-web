# Python AI 教程

欢迎来到面向 AI 的 Python 教程！本教程专为想要进入 AI 领域的学习者设计，采用"面向 AI 场景"的独特教学方法。

## 为什么学习 Python？

Python 是 AI 生态的核心编程语言：

- **AI 生态核心**：几乎所有主流深度学习框架都支持 Python
- **语法简洁**：代码可读性强，开发效率高
- **丰富的库**：NumPy、Pandas、PyTorch 等强大的数据处理和机器学习库
- **社区活跃**：大量开源项目和文档资源

## 学习建议

1. **动手实践**：每个示例都要自己敲一遍
2. **完成练习**：每节练习题都要独立完成
3. **结合场景**：理解每个知识点在 AI 中的应用
4. **循序渐进**：按章节顺序学习，不要跳跃
