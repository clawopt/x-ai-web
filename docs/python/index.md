# Python教程

Python是AI生态的核心编程语言，语法简洁优雅，拥有丰富的标准库和第三方库。

## 为什么学习Python？

- **AI生态核心**：几乎所有主流深度学习框架都支持Python
- **语法简洁**：代码可读性强，开发效率高
- **丰富的库**：NumPy、Pandas、PyTorch等强大的数据处理和机器学习库
- **社区活跃**：大量开源项目和文档资源

## 学习路径

1. **基础语法**：变量、数据类型、控制流、函数
2. **数据结构**：列表、字典、集合、元组
3. **面向对象**：类、继承、多态、封装
4. **标准库**：os、sys、json、re等常用模块
